To use this Linux library with Quisk, enter "make". It supports the NR2 and SNB features.
The resulting libwdsp.so file is used in-place. No installation is necessary.
For Windows, the libwdsp.dll file is included in the quisk directory.

The WDSP library was originally written for Windows by Warren Pratt, NR0V.
John Melton, g0orx/n6lyt, contributed a Linux port.
Christian, DL1YCF, contributed further improvements.
